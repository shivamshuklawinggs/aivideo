# Voice Samples

This folder contains sample voice files for the AI webtoon explainer.

## Available Voice Samples

### Male Voices
- `narrator_male.wav` - Deep male narrator voice
- `character_male.wav` - Male character voice

### Female Voices  
- `narrator_female.wav` - Clear female narrator voice
- `character_female.wav` - Female character voice

## Usage

These voice samples are used by the AI service for:
1. Voice synthesis demonstrations
2. Audio generation for webtoon explanations
3. Default voice profiles for new users

## File Format

All samples are in WAV format for maximum compatibility with the AI processing pipeline.
